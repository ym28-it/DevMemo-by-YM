Log:
WebAssembly C JavaScript
オリジナルのCプログラムをEmscriptenでコンパイルして作りました
システムはHTML・CSS・JavaScript・Wasmで実装されているため、処理はユーザのデバイス内で完結します
LRSはポリトープの頂点計算や凸包計算を行うアルゴリズム
現在はlrs mp, lrs long, lrs hybridの三種類が使用可能
lrs longは64bit 128bitとそれぞれオーバーフローのチェックを行うモードと行わないモードがあります
The original C program was compiled and created in Emscripten
The system is implemented in HTML, CSS, JavaScript, and Wasm, so processing is completed within the user's device
LRS is an algorithm that performs polytope vertex and convex hull calculations
Currently three types are available: lrs mp, lrs long, and lrs hybrid
lrs long is available in 64-bit and 128-bit modes with and without overflow checking, respectively
