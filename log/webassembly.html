WebAssembly

About WebAssembly

実行環境とは独立したWasm Runtime上で動作する
W3Cで標準化されている
コンパイラ：Emscripten
拡張子：.wasm
Runs on Wasm Runtime, which is independent of the execution environment
Standardized by W3C
Compiler: Emscripten
Extension: .wasm

Emscripten

WebAssembly用のコンパイラ
C/C++ Go Rust Zigなどに対応している

Install Emscripten

git clone https://github.com/emscripten-core/emsdk.git
cd emsdk
./emsdk install latest
./emsdk activate latest
source ./emsdk_env.sh
emcc -v
